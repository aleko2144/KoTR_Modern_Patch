# KoTR Modern Patch
Исходники [Modern Patch](https://www.moddb.com/games/hard-truck-2/downloads/king-of-the-road-modern-patch) для игры "Дальнобойщики 2".</br>
Исходники версии v1.00/v1.01 (включая модификации) доступны [здесь](https://github.com/aleko2144/KoTR_Modern_Patch/tree/1.00).

## Что это?
Патч в виде asi-плагина, как исправляющий недоработки оригинальной игры, так и вносящий некоторые улучшения.</br>
Список изменений доступен [здесь](CHANGELOG.md).

## Совместимость с версиями игры
|Исправление|Описание|Поддерживаемые версии игры|
|-----|-----------|-----------------------|
|QueueSortingFix|Исправляет секундную задержку при въезде в Алмазный и сокращает время загрузки игры|6.6 - 8.2|
|ExtCameraFix|Исправляет движение внешней камеры за машиной игрока|6.7 - 8.2|
|DustParticlesFix|Корректирует направление летящих частиц пыли от автомобилей|7.0 - 8.2|
|STrailersPhysFix|Корректирует движение полуприцепов|7.0 - 8.2|
|CollisionsFix|Корректирует параметры обработки столкновений между динамическими объектами (исправляет "залипания" транспорта при столкновениях между собой)|7.2 - 8.2|
|MirrorsFix*|Исправляет проблемы с отрисовкой зеркал при игре в разрешениях формата 16х9|7.2 - 8.2|
|AICarFix|Корректирует поведение трафика на двухполосных дорогах|7.4 - 8.2|
|CabinCamFix|Исправляет ошибку, из-за которой салоны имели заниженное положение|8.0 - 8.2|
|TechSystemFix**|Исправляет ошибки системы улушчений, такие как пропажа топлива при смене автомобиля с установленным дополнительным баком, проблемы с отображением установленных дополнений, и так далее|8.0 - 8.2|

*_Корректно работает только с [D2GI](https://github.com/REDPOWAR/D2GI/), при этом должны быть включёны хуки в ini-файле_</br>
**_Экспериментальная модификация - влияние на игру до конца не изучено_</br>

__Версии 8.x совместимы с европейскими 1.x. Так что патчи для 8.2 работоспособны и на 1.3.__

## Сборка проекта
* Для компиляции текущей версии проекта требуется VS2019, для версий 1.00/1.01 - VS2012.
* Для подгрузки ASI-файла в игру рекомендуется использовать [Ultimate ASI Loader](https://github.com/ThirteenAG/Ultimate-ASI-Loader) за авторством [ThirteenAG](https://github.com/ThirteenAG)
* Модифицированные ресурсы игры для текущей версии патча опубликованы здесь __(только для версий игры 8.x/1.x)__: https://disk.yandex.ru/d/8vJAj0gO_5vukA
